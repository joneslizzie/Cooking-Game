# Cooking-Game

README

Run cells in this order (based on my cell markdown):
1. "Import"
- Run all the imports successfully 
2. "Extract Frames": 
- Extract frames from the video successfully and save them to a "frames" folder, or outputs "frames already exists"
3. "Get Centroids"
- Saves the 2D list of all the centroids to "centroids", prints that list
4. "Run and Create the Game"
- Compiles my game successfully and a popup appears so that you can play

My game:
- use arrow keys to move the cookie jar
- collect 3 cookies in 30 seconds to level up, then collect 5 cookies in 30 more seconds to win
- if you hit the chef you are kciked to the bottom left corner of the game
